Definition

Mathematical Markup Language (MathML) is a mathematical markup language which used for XML to describe mathematical natotions and display both the content and the structure.

Presentation

Presentation MathML

Presentation MathML has around 30 elements which all begin with "m". For example:

  • mi - identifiers
  • mo - operators
  • mn - numbers
  • mtext - text
  • mrow - a horizontal row of items
  • mfrac - fractions
  • msqrt and mroot - roots
  • mfenced - surrounding content with fences
